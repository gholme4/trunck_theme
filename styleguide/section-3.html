3 Buttons

Button styles can be applied to anything with the .button class applied. However, typically you'll want to apply these to only <a> and <button> elements for the best rendering.

3.1 Base styles & states

:hover
Hover state.
:active
Active state.
.disabled
Disabled button.. 
<button type="submit" class="button primary [modifier class]">Button</button>

3.2 Sizes

Add .tiny, .small, or .medium, .large, .expanded for additional sizes.

.tiny
Tiny button.
.small
Small button.
.medium
Medium button.
.large
Large button.
.expanded
Block level button.
.expanded.small
Small block level button. Classes can be combined. 
<button type="submit" class="button primary [modifier class]">Button</button>

3.3 Colors

Add .secondary, .success, .alert, or .warning for additional colors.

.secondary
Secondary color button.
.success
Success color button.
.alert
Alert color button.
.warning
Warning color button. 
<button type="submit" class="button [modifier class]">Button</button>

3.4 Hollow Buttons

Add .hollow.

.secondary
Secondary color button.
.success
Success color button.
.alert
Alert color button.
.warning
Warning color button. 
<button type="submit" class="button hollow [modifier class]">Button</button>

3.5 Dropdown Buttons

Add a dropdown arrow to your button with the .dropdown class.

<button class="button dropdown">Button</button>

3.6 Button Group

Add the .button-group class to a container, and inside it place any number of buttons. The buttons are separated by a small border.

One Two Three 
<div class="button-group">
	<a class="button">One</a>
	<a class="button">Two</a>
	<a class="button">Three</a>
</div>

3.7 Split Button

To build a split button, just create a button group with two buttons. To create a button with only an arrow, add the class .arrow-only. Note that the button still needs a label for screen readers, which can be embedded inside the button with a .show-for-sr element. In the example below, an assistive device will read the arrow button as "Show menu".

Primary Action Show menu 
<div class="button-group">
	<a class="button">Primary Action</a>
	<a class="dropdown button arrow-only">
		<span class="show-for-sr">Show menu</span>
	</a>
</div>
