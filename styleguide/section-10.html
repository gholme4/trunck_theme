10 Reveal

Modal dialogs, or pop-up windows, are handy for prototyping and production. Foundation includes Reveal, our jQuery modal plugin, to make this easy for you.

10.1 Basics

To create a modal, add the class .reveal, the attribute data-reveal, and a unique ID to a container.

Click me for a modal

Reveal modal content

<p><a data-toggle="exampleModal1" aria-controls="exampleModal5">Click me for a modal</a></p>
<div class="reveal" id="exampleModal1" data-reveal>
	<p>Reveal modal content</p>
	<button class="close-button" data-close aria-label="Close reveal" type="button">
		<span aria-hidden="true">&times;</span>
	</button>
</div>

10.2 Sizing

On small screens, a modal is always 100% of the width of the screen. On medium-sized screens and larger, the width changes to 80%.

The size of a modal can be changed with these sizing classes, which are added to the modal container:

  • .tiny: 30% wide
  • .small: 50% wide
  • .large: 90% wide
  • .full: 100% width and height, defaults the escClose option to true, as well as creates a close button.

10.3 Full-screen

A full-screen modal is 100% of the width and height of the window. Add the .full class to make it go.

Click me for a full-screen modal

Reveal modal content

<p><a data-toggle="exampleModal8" aria-controls="exampleModal5">Click me for a full-screen modal</a></p>
<div class="reveal full" id="exampleModal8" data-reveal>
	<p>Reveal modal content</p>
	<button class="close-button" data-close aria-label="Close reveal" type="button">
		<span aria-hidden="true">&times;</span>
	</button>
</div>

10.5 Animations

To use animations from the Motion UI library, include the data-animation-in="someAnimationIn" and data-animation-out="someAnimationOut" attributes.

Click me for a modal with an animation.

Reveal modal content

<p><a data-toggle="exampleModal10" aria-controls="exampleModal10">Click me for a modal with an animation.</a></p>

		<span aria-hidden="true">&times;</span>
