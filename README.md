### Bower
```shell
bower install
```

### NPM
After cloning/downloading, cd to the theme folder and run the following commands to install required node modules and bower packages:
```shell
npm install
```
## Grunt
To compile once run:
```shell
grunt
```
To compile and start watch run:
```shell
grunt dev
```
